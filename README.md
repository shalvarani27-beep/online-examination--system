The Online Examination System is a web-based application that allows students to take exams online and enables administrators to manage exams, questions, and results efficiently.

## Features
- Admin login and exam management
- Student registration and login
- Online exam with timer
- Automatic result calculation
- Secure and user-friendly interface

## Technologies Used
- HTML
- CSS
- JavaScript
- Java (Servlets / JSP)
- MySQL
- Apache Tomcat

## Modules
### Admin
- Add and manage exams
- Create and manage questions
- View student results

### Student
- Register and login
- Attend exams online
- Submit answers
- View results

## How to Run
1. Install Apache Tomcat
2. Configure MySQL database
3. Import project into Eclipse / IntelliJ
4. Deploy on Tomcat server
5. Open browser:
   http://localhost:8080/Online-Examination-System
